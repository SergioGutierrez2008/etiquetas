Etiquetas extras

Etiqueta pre

La etiqueta pre sirve para respetar espacios y líneas en blanco, al contrario de la etiqueta párrafo.

Hola soy Sergio tengo 16 años 

        Hola    soy 
        Sergio      tengo 16 años

Etiqueta abr

La etiqueta abr se utiliza para las abreviaturas-

Según datos de la OMS el consumo excesivo de azúca en niños causa daños severos en la salud.

Etiqueta bdo

Si se desea cambiar la orientación de un texto, se utiliza bdo.

Este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta cite se utiliza para escribir citas textuales

Especialidad haciendo billetes Siempre andamos activos, aquí no paramos jamás, somos jefes por: Jasiel Nuñez

Etiqueta small

La etiqueta small hace el texto más pequeño

Este texto está dentro de una etiqueta small

Etiqueta sub

Si se desea escribir un subíndice, se debe utilizar la etiqueta sub

147478487 Este es un texto diferente

Etiqueta sup

Si se desea escribir con superíndice, se debe utilizar la etiqueta sub

09737887458646456 Pacas verdes duplicadas aqui andamos el negocio está redondo y me la paso en mi camaro ruleteando

Etiqueta time

Para escribir fechas y horas se debe usar la etiqueta time

Mañana es día 2024-11-06

Mañana es día 6 de noviembre del 2024

Etiqueta address

Para especificar al navegador que el texto se refiere a una dirección se utiliza una etiqueta address

Loma de Oro No. 501 Col. Lomas de Rosales CP: 89100 Tampico, Tamaulipas
